# Security Policy

## Supported versions

This repository is a template. Security fixes are applied to the default branch as they are identified.

## Reporting a vulnerability

Do not open public issues for security vulnerabilities.

Report vulnerabilities privately using one of the following options:

- Open a private security advisory in GitHub for this repository (preferred).
- If GitHub advisories are not available, contact the repository owner directly.

Include:

- A clear description of the issue and impact
- Reproduction steps (as safe as possible)
- Any logs or error output (redact secrets)
- Suggested remediation (optional)
